The com.slamd.resourcemonitor package contains the code used to obtain resource monitor statistics. There are a number of components held in this package:
  • The resource monitor client, which handles all interaction with the SLAMD server.
  • The abstract resource monitor API, which defines the methods that must be implemented by resource monitors.
  • A number of concrete resource monitor implementations, including those capable of measuring CPU utilization, memory consumption, network load, and disk I/O.
